
# Do
Lightweight **Schedule Widget** written in C++ for Windows and Linux by yours only, 217eastbroadway.

---
### Features
- Basic Task Scheduling (Insertion, Deletion, Completion)
- Time-Based Scheduling (Task Timers, Set Task to End at Certain Time)
- UI Customizability (UI Font, Textures, etc...)

---

### How To Compile (and run)

Prerequisites (Windows)
- CMake >3.10
- MSVC

Prerequisites (Linux / MacOS)
- Cmake >3.10
- CXX Compiler (GCC / CLang)
- Raylib (must be installed manually on your os)
	- Debian / Ubuntu install
	`sudo apt install libraylib-dev`
	-  RedHat / Fedora install
	`sudo dnf install raylib-devel`
	
Compiling and Running on Windows
`./run.bat`

Compiling and Running on Linux / MacOS
`chmod +x ./run.sh && ./run.sh`

**Done, enjoy!**
